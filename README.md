# self-calib-road (도로 카메라 자동 캘리브레이션)

도로 CCTV 등 고정형/차량용 카메라 영상에서 **차선(SAM3)** 과 **차량 궤적(RT-DETR/BoT-SORT)** 을 이용하여 **체커보드 없이 카메라 내부 파라미터(Intrinsics)**를 자동 추정하는 프로젝트입니다. (외부 파라미터 제외)

## 📌 목표 및 접근 방식

| 파라미터   | 설명                     | 캘리브레이션(추정) 방식             |
| ---------- | ------------------------ | ----------------------------------- |
| `fx`, `fy` | 초점 거리 (Focal Length) | 차선의 소실점(Vanishing Point) 계산 |
| `cx`, `cy` | 주점 (Principal Point)   | 영상 중심점으로 고정 가정           |
| `k1`, `k2` | 방사 왜곡 계수           | 차량 직선 궤적 최적화 (향후 진행)   |
| `p1`, `p2` | 구면(접선) 왜곡 계수     | 차량 직선 궤적 최적화 (향후 진행)   |

### 🔭 프로젝트 확장 단계 (Phases)

**Phase 1: 고정형 카메라 정적 캘리브레이션 (현재)**
- 고정된 방위각/초점거리를 가진 N개의 프레임 통계(소실점 교차 등)를 분석하여, 단일 내부 파라미터(상수) 산출 및 보정.

**Phase 2: PTZ 카메라 동적 캘리브레이션 (예정)**
- 기존 구현된 차선/차량 프레임 단위 추출 코어 로직을 기반으로, 렌즈 줌(Zoom)이나 카메라 회전(Pan/Tilt) 시에도 **실시간 변화하는 내부/외부 파라미터를 추적(Temporal Tracking)** 하는 모듈 개발.
- 칼만 필터(Kalman Filter) 등을 활용해 차선의 기울기와 소실점 이동 폭을 실시간으로 추정하는 Auto-Calibration 파이프라인.

## 🚀 현재까지의 개발 진행 상황 (Features)

**1. SAM3 기반 정밀 차선 검출 및 후처리 (`LaneDetector`)**
- 모델 추론으로 도출된 폴리곤/세그멘테이션 마스크를 **중심선(Center Polyline)** 형태의 1차원 좌표계로 변환.
- **노이즈 필터링 로직 구현:**
  - PCA 제원 분석을 통해 방향성이 없는 블록 형태 노이즈 제거
  - 횡단보도나 과속 방지턱 같은 가로 방향 선형(수평선) 노이즈 제외
  - Y축 기준 길이가 짧은 파편적 노이즈 제거
- **차선 스무딩 (Smoothing):**
  - 추출된 포인트들에 대해 1차 방정식(직선) 피팅을 주어, 차선 끝자락이 꺾이거나 튀는 삑사리 현상 완벽 방지.
- **점선 및 실선 병합 (Merging):**
  - 단순 직진 거리뿐 아니라 **측면(Lateral) 거리 및 종방향 거리**를 분리하여 판별.
  - 이격 거리가 먼 점선이라 하더라도, 같은 방향성을 가지면 하나의 긴 차선으로 병합하는 강력한 Clustering 적용.
- **방향성 그룹화:** 비슷한 각도(15도 이내)로 향하는 병합된 차선들을 동일한 방향성 그룹으로 결속.

**2. RT-DETR 기반 강건한 차량 검출 및 추적 (`VehicleTracker`)**
- `RT-DETR-L` 모델을 활용한 실시간 고정밀 차량 검출(Object Detection).
- **BoT-SORT 알고리즘**을 결합하여 프레임 간 끊김 없는 ID 유지 및 연속적인 궤적(Trajectory) 추출.
- 차량 바운딩 박스의 하단 중앙(Bottom-center) 좌표를 추적하여, 소실점 연산 및 방사 왜곡 계수 최적화에 필요한 기하학적 리소스 제공.

**3. 시각화 및 UI 개발 (`VideoWidget`)**
- PySide6(Qt) 기반 뷰어 구성 (좌표계 변환, 영상 스케일 유지 기능)
- 차량 Bounding Box 및 객체별 궤적(Trajectory) 오버레이 렌더링.
- 차선 랜더링 시 **방향성 그룹(`direction_group`)에 맞춰 같은 진행 방향을 가진 차선을 동일한 색상**으로 시각화하여 사용자의 직관적인 분석을 도움.
